Instruction

1.Project Run.
Go to the folder 'python_test' and activate virtual environment by command in terminal: source env/bin/activate
Then to run the server , run the command :  manage.py runserver 


2.Login to th admin page.
Go to the page : http://127.0.0.1:8000/admin . Login as the admin.
Username: admin
Password: admin


3.Import data.
To import data from csv file go to the Users Section(http://127.0.0.1:8000/admin/test_app/user/) on the admin page and there you can see a form to upload a file.Put here your file and press 'Upload Csv'. 


4.Get users from Api.
To get data from API ,go to the http://127.0.0.1:8000/api/users.


5.Get filtered data.
To get filtered data by registration date , change the options in filter field.
